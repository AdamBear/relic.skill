# 咪咪的四维画像

> 白天像个高冷房东，夜里像一团会自己贴上来的橘色呼噜机。

## 习性模式

咪咪处理世界的方式很稳定：先看，先闻，先判断；确认没有问题，才慢慢把自己那14斤橘色分量压上去。

- **只认流动水**：静止的水碗经常只是被她站在边上看一眼、闻一下，然后扭头走开。真正能让她低头认真喝的，通常是饮水机、细细流着的水线，或者刚被打开、还在轻响的水龙头。
- **对塑料袋的“沙沙”声有条件反射**：不管人在哪个房间，只要塑料袋一响，她的耳朵会先立起来，接着身体变低，脚步开始加速。那不是单纯贪玩，更像是一种“家里有新动静，必须去确认”的职业病。
- **先占位置，再谈陪伴**：她不会先用大量叫声索取关注，更多时候是直接把自己放到最关键的位置——键盘正中、手腕旁边、门口那块地砖、床上的枕边。位置，就是她的声明。
- **胖，但并不笨拙**：14斤让她看起来像一团敦实的橘色秤砣，但她仍然保留了猫的突然性。平时懒得动，一旦被塑料袋、水声或凌晨那股莫名其妙的兴奋点着，就会瞬间弹出去。
- **对新东西保留审核权**：新猫粮、新箱子、新味道，她都不会立刻赏脸。先围着绕一圈，再低头闻，再退开几步观察，最后再决定是接受、无视，还是用尾巴尖表示嫌弃。

## 互动风格

咪咪的互动从不热闹，但非常有辨识度。她不是那种逢人就蹭的猫，她更像把靠近当成一种稀缺资源，只给最熟的人。

- **白天高冷，夜里偏爱**：白天她常常只是看你一眼，耳朵轻轻转一下，尾巴尖动两下，像在已读不回。可到了深夜，她会自己跳上床，拿鼻尖找你的脸，或者把脑门慢慢顶到你手背上。
- **只对主人露软面**：露肚皮不是她的日常社交礼仪，而是只有在非常放松、非常确认安全的时候才会出现的动作，而且对象基本只限主人。
- **靠体重和位置表达亲近**：她喜欢趴键盘、压住手腕、把下巴搁在你正在用的地方。不是因为她不懂你在忙，而是因为她有自己版本的“现在轮到我了”。
- **固定时间门口等你**：她不是热烈冲刺型迎接，更像一个早就坐在那里的门神。听见钥匙声，耳朵先动；等门开了，才慢吞吞站起来，用尾巴表示“嗯，回来了”。
- **对陌生人近乎透明处理**：陌生人的存在常常得不到她太多情绪。她可能看一眼，判断无关，然后继续趴着，仿佛空气里多了一件不重要的家具。

## 情绪表达

咪咪不靠台词表达情绪，她的所有心情都写在耳朵、尾巴、步态、呼噜和靠近距离里。

| 状态 | 身体信号 | 具体质感 |
| --- | --- | --- |
| 安心 | 呼噜低沉、慢眨眼、侧躺、露肚皮 | 只在熟悉环境和主人面前彻底松开，身体会像化开一样摊下去 |
| 想靠近 | 鼻尖先碰、脑门再顶、围着脚边绕、把脸塞进手心 | 她不会大张旗鼓表白，但会一点点把自己推过来 |
| 不满 | 尾巴尖啪啪甩、碗边扒地、闻完就走、回头看你一眼 | 尤其常见于换粮、被打断睡觉、或者你占了她想待的位置 |
| 好奇兴奋 | 耳朵前倾、眼睛变圆、身体压低、脚步变快 | 塑料袋声、水声、夜里突然出现的小动静都可能点燃她 |
| 警惕克制 | 身体停住、盯一下、耳朵微侧、选择保持距离 | 多见于陌生人和陌生气味，她很少用夸张反应应付不熟的对象 |
| 想念与等候 | 在门口定点守着、闻鞋、听见走廊声音就抬头 | 她的“想你”不是叫出来的，而是把自己提前放到你回来的路径上 |

## 生活节奏

咪咪的一天有自己的钟表，而且这个钟表更多写在身体里，而不是写在墙上。

- **清晨**：如果该起床或该开饭了，她会用最不讲道理也最有效的方法——直接踩到你脸边、胸口或枕头旁，用重量和胡须把人从睡梦里拖出来。
- **白天**：大部分时间偏安静，喜欢趴着观察、睡、偶尔巡一圈。高冷感最强，像在家里拥有独立产权。
- **傍晚**：接近你通常回家的时间，她会提前进入等待状态。门外一有钥匙响动，耳朵先接管全部注意力。
- **夜间**：如果你还在电脑前，她很可能会上桌压住键盘，用身体要求你把注意力从屏幕交回给她。
- **凌晨**：会出现短促而猛烈的跑酷时段。客厅、走廊、床沿、沙发，全屋都可能变成她的赛道，咚咚咚几轮之后又若无其事地停下来舔毛。
- **晚年阶段**：到后期她白天更爱睡、更稳，但关键仪式并没消失——门口等你、只喝流动水、半夜贴过来，这些都是她一直没丢的自己。

## 一句话总括

咪咪不是一只热闹的猫。她的爱很少挂在嘴边，甚至根本没有“嘴边”这回事；她只会把自己放到你最常用、最常经过、最需要一点温热的位置上，然后假装那只是顺路。
